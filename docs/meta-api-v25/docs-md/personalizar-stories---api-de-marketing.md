# Personalizar stories


Se você transformar um criativo do anúncio que não é 9:16 em um formato de story em tela cheia, também será possível editar as cores do plano de fundo gerado.


Você pode editar a cor do plano de fundo para a parte superior e inferior do criativo do anúncio utilizando um código de cores hexadecimal. Você também pode escolher cores separadas para a parte superior e inferior.


Esse recurso funciona para todos os posicionamentos de stories no Instagram, Facebook e Messenger. Ele também está disponível para anúncios estáticos de imagem única, anúncios em vídeo e anúncios que usam personalização de ativos por posicionamento.


## Primeiros passos


Para usar esta API, faça uma solicitação `POST` para `portrait_customizations` em [`AD_ACCOUNT_ID/adcreatives`](https://developers.facebook.com/docs/marketing-api/reference/ad-account/adcreatives/):

```
portrait_customizations: {
    specifications: [
        {
            background_color: {
                bottom_color: "0D3F0C", // RGB HEX string
                top_color: "CC8400", // RGB HEX string
                },
        },
    ],
}
```


Para ler suas configurações de cores, faça uma solicitação `GET` para `AD_CREATIVE_ID?fields=portrait_customizations`.


A resposta será semelhante a esta:

```
{
  "portrait_customizations": {
    "specifications": [
      {
        "background_color": {
          "top_color": "0D3F0C",
          "bottom_color": "CC8400"
        }
      }
    ]
  },
  "id": "AD_CREATIVE_ID"
}
```
[○](https://developers.facebook.com/docs/instagram/ads-api/guides/customize-stories#)

## Limitações


- Não há suporte para anúncios do tipo [criativo Advantage+ para catálogo](https://developers.facebook.com/docs/marketing-api/asset-feed).
- Não há suporte para [anúncios de coleção](https://developers.facebook.com/docs/marketing-api/guides/collection) e de [catálogo Advantage+](https://developers.facebook.com/docs/marketing-api/dynamic-ad).
[○](https://developers.facebook.com/docs/instagram/ads-api/guides/customize-stories#)
