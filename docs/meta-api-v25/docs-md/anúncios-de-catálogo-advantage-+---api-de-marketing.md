# Anúncios de Catálogo Advantage+


Com os [Anúncios de Catálogo Advantage+](https://www.facebook.com/business/a/online-sales/dynamic-product-ads), é possível promover itens relevantes de um catálogo em qualquer dispositivo. Mostre anúncios de milhares de itens para o público certo e automatize o processo. Essa solução tem três elementos principais:


- [Criação de um feed para o catálogo](https://developers.facebook.com/docs/marketing-api/catalog) a fim de veicular imagens, descrições e preços nos anúncios de forma automática.
- Configuração de [Eventos do App](https://developers.facebook.com/docs/app-events) ou do [Pixel da Meta](https://developers.facebook.com/docs/meta-pixel) para mensurar ações (como compras) e definir o perfil dos [públicos-alvo](https://developers.facebook.com/docs/marketing-api/audiences/guides/dynamic-product-audiences) de um anúncio.
- [Criação e veiculação de anúncios com base no seu catálogo](https://developers.facebook.com/docs/marketing-api/dynamic-ads/get-started) para mostrar itens relevantes do feed do catálogo e definir o público-alvo.


Depois de configurar esses elementos, é possível ver como as pessoas no Facebook estão [interagindo com os anúncios](https://developers.facebook.com/docs/marketing-api/reference/adgroup/insights/), bem como usar [ferramentas de depuração](https://developers.facebook.com/docs/marketing-api/dynamic-product-ads/debugging-tools) para diagnosticar e resolver problemas.
