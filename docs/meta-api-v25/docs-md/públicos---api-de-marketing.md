# Públicos


O direcionamento de público ajuda a exibir seus anúncios para as pessoas que interessam a você. Existem duas abordagens gerais (específica ou ampla) para a criação de um público-alvo. Escolha uma dessas opções com base no seu objetivo e nos recursos disponíveis.


É possível usar uma abordagem mais específica e criar públicos com base em dados do cliente, dados de conversão (como atividade no app ou site) e assim por diante. Outra opção é fornecer informações mais amplas, como dados demográficos ou localização, para que possamos exibir seus anúncios às pessoas que atendem a esses atributos.


## Usos comuns


- [**Públicos semelhantes**](https://developers.facebook.com/docs/marketing-api/lookalike-audience-targeting): direcione os anúncios a pessoas com características semelhantes às dos seus clientes já estabelecidos.
- **Públicos personalizados**: crie um público-alvo personalizado com dados de comportamento no [**app para celular**](https://developers.facebook.com/docs/marketing-api/audiences-api/mobile-apps) e do [**site**](https://developers.facebook.com/docs/marketing-api/audiences-api/websites), do [**CRM**](https://developers.facebook.com/docs/marketing-api/audiences-api) e de [**sinais de engajamento**](https://developers.facebook.com/docs/marketing-api/audiences-api/engagement). Também é possível criar públicos a partir de [**conversões offline**](https://developers.facebook.com/docs/marketing-api/audiences-api/offline).
- [**Públicos dinâmicos**](https://developers.facebook.com/docs/marketing-api/dynamic-product-ads/product-audiences): crie um público a partir de sinais do app para celular e do site.
- [**Opções de direcionamento**](https://developers.facebook.com/docs/marketing-api/buying-api/targeting): o direcionamento básico inclui dados demográficos e eventos, localização, interesses e comportamentos. Você também pode aprender sobre [**direcionamento avançado**](https://developers.facebook.com/docs/marketing-api/targeting-specs).
