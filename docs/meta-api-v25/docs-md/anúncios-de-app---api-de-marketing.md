# Anúncios de app no Threads


Os [anúncios de app](https://developers.facebook.com/docs/marketing-api/mobile-app-ads/) estão disponíveis para o Threads. Para veicular no Threads, inclua `instagram` e `threads` em `publisher_platforms` no seu conjunto de anúncios. Depois, use o posicionamento `stream`. Não se esqueça de selecionar também o posicionamento `stream` do Instagram.


Para fornecer um criativo para anúncios de app do Threads, inclua `instagram_user_id` e `threads_user_id` como parte do parâmetro `object_story_spec` do criativo.


### Limitações


- O Threads é compatível com anúncios de instalação de app para celular e de engajamento com o app. [Anúncios de instalação do app para computador](https://developers.facebook.com/docs/marketing-api/mobile-app-ads) não são compatíveis.
- Anúncios de app com imagem, vídeo e carrossel são aceitos. Não há suporte para anúncios interativos. Os anúncios de app interativos serão renderizados como anúncios em vídeo no Threads.


## Antes de começar


- Verifique se você conhece o processo básico de [criação de anúncios no Threads](https://developers.facebook.com/docs/marketing-api/ad-creative/threads-ads/creation).
[○](https://developers.facebook.com/docs/marketing-api/ad-creative/threads-ads/creation/app-ads#)

## Criar um anúncio de app do Threads


### Exemplo de solicitação


O `threads_user_id` e o `instagram_user_id` estão incluídos em `object_story_spec` da solicitação abaixo, permitindo que o anúncio seja veiculado no Threads.

```
curl -X POST \ -F 'name="Sample App Ads Creative"' \ -F 'object_story_spec={ "page_id": "<PAGE_ID>", "instagram_user_id": "<INSTAGRAM_USER_ID>" "threads_user_id": "<THREADS_USER_ID>" "link_data": { "call_to_action": { "type": "INSTALL_MOBILE_APP", "value": { "link": "<APP_STORE_URL>" } }, "image_hash": "<IMAGE_HASH>", "link": "<APP_STORE_URL>", "message": "<MESSAGE_TEXT>" } }' \ -F 'access_token=<ACCESS_TOKEN>' \ https://graph.facebook.com/v25.0/act_<AD_ACCOUNT_ID>/adcreatives
```


### Exemplo de resposta


A resposta bem-sucedida é a nova identificação do criativo do anúncio de app.

```
{
  "id":"<CREATIVE_ID>"
}
```
