# Primeiros passos para usar a API de Marketing


Para usar a API de Marketing com eficiência, os usuários devem seguir algumas etapas para configurar o ambiente e ter acesso aos recursos da API. Esta seção descreve os pré-requisitos necessários.


## Requisitos para contas de anúncios


Para gerenciar os anúncios por meio da API de Marketing, é necessário ter uma [conta de anúncios ativa](https://www.facebook.com/business/help/915885887059947). Essa conta é fundamental não só para veicular campanhas, mas também para gerenciar configurações de cobrança e definir limites de gastos. Na conta de anúncios, você monitora suas despesas com publicidade, acompanha o desempenho e otimiza as campanhas de forma eficaz.


### Como localizar o número da sua conta de anúncios


Para localizar o número da sua conta de anúncios, acesse o [Gerenciador de Anúncios da Meta](https://adsmanager.facebook.com/).


- **Entre no Facebook:** comece entrando na conta no Facebook associada à sua empresa.
- **Acesse o Gerenciador de Anúncios:** é possível encontrar o Gerenciador de Anúncios no menu suspenso no canto superior direito da página inicial ou página comercial do Facebook.
- **Localize sua conta de anúncios:** no Gerenciador de Anúncios, clique nas Configurações da conta de anúncios no menu no canto inferior esquerdo da tela.
- **Visualize as informações da conta de anúncios:** na tela Configurações, você encontrará o número da sua conta de anúncios listado, com outros detalhes, como informações de cobrança e limites de gastos.
[○](https://developers.facebook.com/docs/marketing-api/get-started#)
