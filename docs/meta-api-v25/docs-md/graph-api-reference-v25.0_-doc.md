```
Host: graph.facebook.com
```

### Parameters

This endpoint doesn't have any parameters.

### Fields


| Field | Description |
| --- | --- |
| id numeric string | The document ID |
| can_delete bool | Boolean, can document be deleted by viewer Default |
| can_edit bool | Boolean, can document be edited by viewer Default |
| created_time datetime | Timestamp of when the document was created Default |
| embedded_urls list\<string\> | A list of urls that were embedded into the note |
| from User | Author of the document Default |
| icon string | Image file for document icon Default |
| link string | URL for the document Default |
| message string | Content of the document Default |
| revision id | Current revision ID Default |
| subject string | Title of the document Default |
| updated_time datetime | Last updated timestamp Default |


### Error Codes


| Error | Description |
| --- | --- |
| 100 | Invalid parameter |
