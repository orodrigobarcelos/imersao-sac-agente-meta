# Anúncios de mensagem


Este guia explica como criar e publicar diferentes anúncios de mensagem usando a API de Marketing.


Os anúncios de mensagem direcionam as pessoas que clicam neles diretamente para conversas com sua empresa no Messenger, no Instagram e/ou no WhatsApp. Eles são compatíveis com mensagens de imagem, vídeo, carrossel ou apresentação multimídia. Também é possível incluir um comando interativo para ligação telefônica ou mais de um modelo de mensagem nesses anúncios.


Esses anúncios podem ser usados para alcançar pessoas em grande escala e fornecer serviço individualizado e com destaque.


## Tipos de anúncio


### [Anúncios de clique para o Messenger](https://developers.facebook.com/docs/marketing-api/ad-creative/messaging-ads/click-to-messenger)


Os anúncios de clique para o Messenger direcionam as pessoas que clicam neles diretamente para conversas com sua empresa no Messenger.


#### Geração de lead com anúncios de clique para o Messenger


Com os anúncios de clique para o Messenger destinados à geração de leads, as empresas podem gerar e qualificar leads por meio de um fluxo automatizado de perguntas e respostas no Messenger usando um modelo de conversa automatizada. Esse modelo cria um anúncio de clique para o Messenger, permitindo que sua empresa qualifique leads e continue a conversa com pessoas interessadas nos seus serviços. As empresas podem identificar as preferências do cliente e criar perguntas de desqualificação para priorizar os melhores leads.


#### Anúncios de clique para o Messenger com extensões de produto


As extensões de produto (recurso "Mostrar produto" no Gerenciador de Anúncios da Meta) é uma otimização do criativo Advantage+ que exibe produtos do seu catálogo abaixo da mídia principal nos seus anúncios. São habilitadas em anúncios de clique para o Messenger.


#### Anúncios de clique para assinar


Os anúncios de clique para assinar podem ajudar você a alcançar novos clientes interessados em receber mensagens de marketing no Messenger. As mensagens de marketing (anteriormente "notificações recorrentes") permitem que você se conecte e interaja com seus clientes em conversas automatizadas e contínuas. Quando você configura um anúncio com modelo de mensagem de marketing, as pessoas que clicam nele veem um convite para aceitar o recebimento de mensagens desse tipo. Depois que elas aceitarem, você poderá enviar mensagens de marketing uma vez por dia. Isso pode ajudar a aumentar o engajamento do seu conteúdo ou impulsionar as vendas do seu produto.


**Observação**: se um cliente não confirmar o aceite, ele não receberá mensagens de marketing.


### [Anúncios de clique para o Instagram](https://developers.facebook.com/docs/marketing-api/ad-creative/messaging-ads/click-to-instagram)


Os anúncios de clique para o Instagram Direct direcionam as pessoas que clicam neles diretamente para conversas com sua empresa no Instagram Direct.


### [Anúncios de clique para o WhatsApp](https://developers.facebook.com/docs/marketing-api/ad-creative/messaging-ads/click-to-whatsapp)


Os anúncios de clique para o WhatsApp direcionam as pessoas diretamente para conversas com sua empresa no WhatsApp.


### [Anúncios de clique com vários destinos](https://developers.facebook.com/docs/marketing-api/ad-creative/messaging-ads/click-to-multidestination)


Os anúncios de clique com vários destinos enviam as pessoas que clicam neles diretamente para conversas com sua empresa no app ou nos apps de mensagem (Messenger, Instagram ou WhatsApp) em que elas têm maior probabilidade de responder.
