# Público Semelhante Advantage


Para obter mais informações sobre o público semelhante Advantage, consulte [Central de Ajuda: Sobre o público semelhante Advantage](https://www.facebook.com/business/help/1212225059146059).


**Nota:** campanhas com objetivos compatíveis são habilitadas por padrão.


- Para aceitar, defina o parâmetro `lookalike` dentro de `targeting_relaxation_types` para `1`.
- Para recusar, defina o parâmetro `lookalike` dentro de `targeting_relaxation_types` para `0`.


Se usar o parâmetro `lookalike` dentro de `targeting_relaxation_types` para um objetivo não compatível, será exibida uma mensagem de erro.


### Exemplo


```
curl \
  -F "name=relaxation null test" \
  -F "promoted_object={'application_id': '<APP_ID>', 'object_store_url': 'https://itunes.apple.com/app/<ID>'}" \
  -F "is_autobid=true" \
  -F "daily_budget=100" \
  -F "billing_event=LINK_CLICKS" \
  -F "campaign_id=<CAMPAIGN_ID>" \
  -F "access_token=<ACCESS_TOKEN>" \
  -F "targeting={'geo_locations': {'countries': ['US', 'GB']}, 'user_os': ['iOS'],   'user_device': ['iPad', 'iPhone', 'iPod'], 'custom_audiences': [{<LOOKALIKE_DATA>}],  'targeting_relaxation_types': {'lookalike': 1}}" \
  https://graph.facebook.com/v25.0/act_<AD_ACCOUNT_ID>/adsets
```
