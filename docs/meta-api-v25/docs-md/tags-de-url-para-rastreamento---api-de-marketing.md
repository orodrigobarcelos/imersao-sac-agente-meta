# Tags de URL para rastreamento


Você pode acompanhar o desempenho de uma campanha com uma ferramenta externa usando as `url_tags` do [criativo do anúncio](https://developers.facebook.com/docs/marketing-api/reference/ad-creative) para distinguir diferentes objetos.


É possível fornecer suas próprias tags personalizadas no formato `key=value`, em que `key` e `value` podem ser definidos por você. Esses valores são combinados pelo caractere `&`. Por exemplo, você pode ter algo semelhante a `utm_source=fb&ad=my_ad,`, que pode ser configurado usando:

```
...
$creative->setData(array(AdCreativeFields::URL_TAGS => 'utm_source=fb&ad=my_ad',));

$creative->create();
```


O Facebook também fornece macros especiais que podem ser usadas na posição `value` do formato acima. Para que o nome do anúncio seja preenchido de modo automático, você pode usar a macro `{{ad.name}}`, que será substituída dinamicamente pelo nome quando o anúncio for exibido. Por exemplo, você cria um anúncio chamado `summer_2019` e define a tag `url` como `ad_name={{ad.name}}`. Quando o link do anúncio for renderizado, o usuário verá `ad_name=summer_2019`.

```
...
$creative->setData(array(AdCreativeFields::URL_TAGS => 'ad_name={{ad.name}}',));

$creative->create();
```


Você pode usar algumas macros diferentes para acompanhar suas campanhas:


- `{{campaign.id}}`
- `{{adset.id}}`
- `{{ad.id}}`
- `{{campaign.name}}`
- `{{adset.name}}`
- `{{ad.name}}`


Os parâmetros baseados em nome têm funcionalidade especial. Quando você publica seu objeto pela primeira vez, armazenamos os valores de nome usados em um instantâneo. Se um parâmetro baseado em nome for usado nas tags desse objeto, o Facebook utilizará os valores do instantâneo em vez dos atuais.
