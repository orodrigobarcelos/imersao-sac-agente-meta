# Direcionamento Detalhado Advantage


Para obter mais informações sobre o Direcionamento Detalhado Advantage, consulte [Central de Ajuda: Sobre o Direcionamento Detalhado Advantage](https://www.facebook.com/business/help/128066880933676).


**Nota:** campanhas com objetivos compatíveis não são habilitadas por padrão.


- Para aceitar, defina o parâmetro `targeting_optimization` para `expansion_all`.
- Para recusar, defina o parâmetro `targeting_optimization` para `none`.


Se utilizar o parâmetro `targeting_optimization` para um objetivo não compatível, será exibida uma mensagem de erro.


### Exemplo


```
curl \
  -F "name=relaxation null test" \
  -F "promoted_object={'application_id': '<APP_ID>', 'object_store_url': 'https://itunes.apple.com/app/id1111111'}" \
  -F "is_autobid=true" \
  -F "daily_budget=100" \
  -F "billing_event=LINK_CLICKS" \
  -F "campaign_id=<CAMPAIGN_ID>" \
  -F "access_token=<ACCESS_TOKEN>" \
  -F "targeting={'geo_locations': {'countries': ['US', 'GB']}, 'user_os': ['iOS'], 'user_device': ['iPad', 'iPhone', 'iPod'], 'targeting_optimization': 'expansion_all'}" \
  https://graph.facebook.com/v25.0/act_<AD_ACCOUNT_ID>/adsets
```
