# Registro de alterações


| A versão mais recente da Graph API é: | `v25.0` |
| --- | --- |


Este registro documenta as mudanças [com controle de versão](https://developers.facebook.com/docs/marketing-api/marketing-api-changelog#versioned) e as alterações [fora de ciclo](https://developers.facebook.com/docs/marketing-api/marketing-api-changelog#outofcycle) na API de Marketing.


### Alterações com controle de versão


As alterações com controle de versão são mudanças incluídas com o lançamento de uma nova [versão](https://developers.facebook.com/docs/apps/versions) da API. Em geral, essas alterações são imediatamente aplicadas à versão mais recente e, com frequência, são aplicadas a outras versões em data futura. O registro de alterações que acompanha cada lançamento indica quais mudanças se aplicam à versão atual e quais se aplicam às outras.


Consulte o [guia de atualização](https://developers.facebook.com/docs/apps/upgrading) se quiser saber como atualizar para uma nova versão da API.


### Alterações fora de ciclo


As alterações fora de ciclo são mudanças implementadas fora da nossa programação normal de lançamentos com controle de versão. Dessa forma, não costumam ser aplicadas a uma versão específica. Em vez disso, essas alterações costumam ser aplicadas a todas as versões da API.


O conteúdo das alterações fora de ciclo, que são registradas a cada 15 dias, é gerado automaticamente com base nas alterações da API e é refletido nos arquivos exibidos na pasta [API spec](https://l.facebook.com/l.php?u=https%3A%2F%2Fgithub.com%2Ffacebook%2Ffacebook-business-sdk-codegen%2Ftree%2Fmain%2Fapi_specs%2Fspecs&h=AT7UzdEBJqDGDoijZBdIz5rZNJ3-GQFdIMtv1csu-ZP_A1c0JO1LMXNYrTqaBiO9J0GBN34Dd4Z0-gUKvfrUGL8jdqKhAMhgYLAjoNcBzhTm0lb0Xx3_kOvwJ_TEaVbTX_9_ZEpEybzZtVMP12A6Wyb2zzE).
