# API de Estimativa de Alcance


Obtenha uma variedade de indivíduos para direcionamento e saiba quais lances fazer para alcançar esse grupo-alvo com a API de `/reachestimate`. É possível chamar essa API de uma destas maneiras:


- Indique uma conta de anúncios e uma especificação de direcionamento:

```
curl -G \
  --data-urlencode 'targeting_spec={
    "geo_locations": {"countries":["US","GB"]},
    "publisher_platforms": ["instagram"],
    "user_os": ["iOS"]
  }' \
  -d 'optimize_for=IMPRESSIONS' \
  -d 'access_token=<ACCESS_TOKEN>' \
https://graph.facebook.com/v25.0/act_<AD_ACCOUNT_ID>/reachestimate
```
