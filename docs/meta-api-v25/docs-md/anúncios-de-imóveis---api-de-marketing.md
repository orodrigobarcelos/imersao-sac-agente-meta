# Anúncios de imóveis


Use os anúncios de imóveis do Facebook para impulsionar os sinais de intenção em todos os dispositivos, promovendo automaticamente os classificados relevantes do seu inventário com um criativo único na plataforma. Além disso, use [esta lista](https://www.facebook.com/business/help/224411658455245) para conhecer nossos parceiros de classificado do Facebook Marketplace e entre em contato com eles para promover seu inventário. No momento, esse recurso está disponível apenas em alguns países.


Para usar os anúncios de imóveis do Facebook, você precisa de um catálogo apropriado que inclua propriedades para anunciar, como apartamentos, condomínios, casas, terrenos, entre outros. Cada propriedade tem informações para criar anúncios personalizados. Você pode atualizar os catálogos regularmente usando diferentes opções. Caso você faça carregamentos em lote, também poderá usar uma opção de carregamento parcial que permite criar ou atualizar itens que foram alterados. Veja [Carregamento em lote](https://developers.facebook.com/docs/marketing-api/dynamic-product-ads/product-catalog#direct-upload-product-feed). Essa opção está disponível para todos os Anúncios de Catálogo Advantage+.

O Facebook tem o compromisso de proteger as pessoas contra discriminação, e temos melhorado continuamente a nossa capacidade de detectar e deter potenciais abusos. A discriminação ao direcionar injustamente ou excluir grupos específicos de pessoas é uma violação das [nossas políticas](https://www.facebook.com/policies/ads/prohibited_content/discriminatory_practices). Como parte de um [acordo de compensação histórica](https://l.facebook.com/l.php?u=https%3A%2F%2Fnewsroom.fb.com%2Fnews%2F2019%2F03%2Fprotecting-against-discrimination-in-ads%2F&h=AT62Ah1Lvv6q3mbfa32mConLyod4b8hYT-v1uLCFEc_1uCDojcsySq9w6QqhVoxGj6brFNmHOcup8hzyl4KxI8hrV7s1Ps0kDpqmRouSh-mMAQZ4xG-j-nW8v2kV-R56iDYRohn46030W38PgX8hO2wqSscq82KirRwdoEYb), estamos fazendo mudanças na forma como gerenciamos os anúncios de imóveis.


Os anunciantes precisam especificar `HOUSING` como uma `special_ad_category` para campanhas publicitárias que comercializam imóveis. Ao fazer isso, o conjunto de opções de direcionamento disponíveis para anúncios nessas campanhas será restringido. Os [Anúncios de Catálogo Advantage+](https://developers.facebook.com/docs/marketing-api/dynamic-product-ads/ads-management) que usam catálogos de classificado de imóveis precisam respeitar essas restrições. Leia [Categoria de anúncio especial](https://developers.facebook.com/docs/marketing-api/special-ad-category) para saber mais.
