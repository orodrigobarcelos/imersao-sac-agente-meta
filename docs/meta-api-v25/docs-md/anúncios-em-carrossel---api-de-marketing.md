# Anúncios em carrossel do Instagram


Para criar [anúncios em carrossel](https://developers.facebook.com/docs/marketing-api/guides/carousel-ads/), forneça um criativo do anúncio com vários `child_attachments` em `link_data` para o stream do Instagram. São necessários pelo menos três `MOBILE_APP_INSTALLS` ou `MOBILE_APP_ENGAGEMENT` e dois para outros objetivos. Você precisa fornecer o `instagram_user_id` e usar apenas `objectives` compatíveis com anúncios do Instagram.


Há algumas diferenças entre os anúncios em carrossel do Facebook e do Instagram. Elas só acontecem com o posicionamento do Instagram. Se você tiver um conjunto de anúncios com posicionamentos do Facebook e do Instagram habilitados, o que funciona no Facebook pode ser diferente no Instagram. Por exemplo, o posicionamento `multi_share_optimized` não funciona no Instagram, mas funciona no Facebook.


- O `multi_share_optimized` não funciona no Instagram. Se mais de cinco anexos forem fornecidos, os cinco primeiros serão usados, com base na ordem em que forem inseridos na criação do criativo do anúncio.
- O `multi_share_end_card` não funciona no Instagram. Os cartões finais não estão disponíveis no Instagram.
- Cada anexo precisa ter o `link` especificado, que é o destino do clique na imagem.
- Cada anexo tem um botão de chamada para ação. Se isso não for especificado, o Instagram usará o padrão "Saiba mais" e direcionará para a configuração de `link` do anexo.
- O campo `caption` pode ser usado para especificar o URL de exibição. Caso isso não seja especificado, o URL `link` será usado.
- Cada cartão do carrossel tem uma imagem ou vídeo e um botão de chamada para ação. Além disso, há uma mensagem da configuração de `message` em `link_data`. O `name` de cada anexo também é exibido, entre a imagem ou o vídeo e a mensagem. O campo `description` de `link_data` e de cada anexo derivado é ignorado.


## Cartões de imagem


- Cada anexo precisa conter um conjunto de `picture` ou de `image_hash`. Não definimos a imagem de `link_data` como padrão.
- É possível fornecer um corte de imagem de `100x100` para cada anexo. Caso contrário, será realizado o corte automático. A imagem final deve ter pelo menos 600 x 600 pixels, o que é significativamente maior do que o requisito mínimo dos anúncios em carrossel do Facebook.
[○](https://developers.facebook.com/docs/instagram/ads-api/guides/carousel-ads#)

## Cartões de vídeo


- O vídeo precisa ser quadrado, com pelo menos 600 px em cada dimensão, no máximo 60 segundos.
- É necessária uma imagem em miniatura. É recomendável usar imagens em miniatura com a mesma dimensão que os vídeos, mas isso não é exigido.
- O vídeo é especificado por `video_id` em um `child_attachment`. A imagem em miniatura é especificada por `image_hash` ou `picture` em um `child_attachment`.
- A imagem em miniatura será exibida no carregamento ou quando o anúncio for exibido com uma pequena parcela visível. O vídeo será iniciado automaticamente e em loop.
[○](https://developers.facebook.com/docs/instagram/ads-api/guides/carousel-ads#)

## Stories


Para o Instagram Stories, crie um carrossel com até 10 imagens ou vídeos. Por padrão, o número de cartões é personalizado e varia com o objetivo de otimizar o desempenho. É possível que um usuário veja um, dois ou três cartões antes de visualizar a opção "Expandir story". As limitações do carrossel no Stories variam conforme suas escolhas quanto ao posicionamento dos anúncios.


Para o posicionamento independente do Instagram Stories, é possível escolher um número fixo de cartões, com um máximo de 3 cartões exibidos antes da opção de expandir o story. Esse recurso é configurado dentro do `carousel_delivery_mode`. Há duas opções no campo:


- `optimal_num_cards`: opção padrão. O número de cartões exibido ao usuário é personalizado automaticamente.
- `fixed_num_cards`: selecionado caso o anunciante queira que um número fixo de cartões seja exibido.


O uso de um número fixo de cartões pode aumentar a quantia que um anunciante paga por resultados.


Outras diferenças de posicionamento:


|  | Independente | Posicionamentos automáticos ou mistos |
| --- | --- | --- |
| Taxa de proporção | 9:16 e diferentes de 9:16. A taxa de proporção mista não é compatível. | Não use o formato 9:16. |
| Mídia compatível | Imagens, vídeos de até 120 segundos (por cartão), fotos e vídeos mistos Se você escolheu um número fixo de cartões, são compatíveis os vídeos de até 15 segundos (por cartão). | Imagens, vídeos de até 120 segundos (por cartão), fotos e vídeos mistos |
| Objetivos compatíveis | Reconhecimento da marca Alcance Tráfego (cliques no link) Mensagens Geração de leads Conversões Instalações do app Visualizações do vídeo Tráfego para o estabelecimento Vendas do catálogo Observação: a opção de exibir um número fixo de cartões não é compatível com as vendas do catálogo. | Reconhecimento da marca Alcance Tráfego (cliques no link) Mensagens Geração de leads Conversões Instalações do app Tráfego para o estabelecimento Vendas do catálogo |


Por exemplo, para criar um carrossel no Instagram Stories:

```
curl \ -F 'name=Sample Creative' \ -F 'object_story_spec={ "instagram_user_id": <IG_USER_ID>, \ "link_data": { "child_attachments": [ { "description": "$8.99", "image_hash": "<IMAGE_HASH>", "link": "https:\/\/www.link.com\/product1", "name": "Product 1", "video_id": "<VIDEO_ID>" }, { "description": "$9.99", "image_hash": "<IMAGE_HASH>", "link": "https:\/\/www.link.com\/product2", "name": "Product 2", "video_id": "<VIDEO_ID>" }, { "description": "$10.99", "image_hash": "<IMAGE_HASH>", "link": "https:\/\/www.link.com\/product3", "name": "Product 3" } ], "link": "<URL>" }, \ "page_id": "<PAGE_ID>" \ }' \ -F 'access_token=<ACCESS_TOKEN>' \ https://graph.facebook.com/<LATEST_API_VERSION>/act_<AD_ACCOUNT_ID>/adcreatives
```


Para criar um carrossel com `fixed_num_cards`:

```
curl -X POST \ -F 'object_story_spec={ "instagram_user_id": "<IG_USER_ID>", "page_id": "<PAGE_ID>", "link_data": { "child_attachments": [ { "link": "<LINK>" "image_hash": "<IMAGE_HASH>", "image_crops": [], }, { "link": "<LINK>", "image_hash": "<IMAGE_HASH>", "image_crops": {} }, { "link": "<LINK>", "image_hash": "<IMAGE_HASH>", "image_crops": {} } ], "message": "<MESSAGE>", "multi_share_end_card": false, "multi_share_optimized": false, "picture": "<PICTURE>" } } ' \ -F 'portrait_customizations={ carousel_delivery_mode: "fixed_num_cards" }' \ -F 'access_token=<ACCESS_TOKEN>' \ https://graph.facebook.com/<LATEST_API_VERSION>/act_<AD_ACCOUNT_ID>/adcreatives
```


Os elementos da experiência instantânea **não são compatíveis** com o formato de anúncio em carrossel.
