# Anúncios de catálogo Advantage+ no Instagram


Os [anúncios de catálogo Advantage+](https://developers.facebook.com/docs/marketing-api/dynamic-ads) estão disponíveis para o Instagram. Você pode usar o direcionamento de públicos personalizados para alcançar pessoas que visitaram ou que se engajaram com seu site.


Para criar um [criativo modelo](https://developers.facebook.com/docs/marketing-api/dynamic-ads/get-started#adtemplate) para [anúncios de catálogo Advantage+](https://developers.facebook.com/docs/marketing-api/dynamic-product-ads) ou [anúncios de viagem](https://developers.facebook.com/docs/marketing-api/dynamic-ads-for-travel), defina `template_data` em `object_story_spec`. Oferecemos compatibilidade com anúncios de catálogo Advantage+ no Feed, nos Stories e em Explorar do Instagram.


## Primeiros passos


A imagem padrão é especificada pelo `image_url` de cada produto ou outras imagens em `additonal_image_urls`. Se `additional_image_index` estiver definido no modelo de anúncio, o criativo precisará ter 600 px ou mais, tanto em largura como em altura. Se a imagem não for quadrada, ela será cortada quando exibida.


`name`, que só pode ser usado com o formato de carrossel, e `message` para `template_data` estão sob a imagem do produto. A `description` não aparece no Instagram, somente no Facebook.


É opcional usar `call_to_action` para `template_data`. Se não for fornecido, um botão **Saiba mais** será criado por padrão com o URL do produto mostrado.
[○](https://developers.facebook.com/docs/instagram/ads-api/guides/advantage-catalog-ads#)

## Limitações


Os anúncios de catálogo Advantage+ para o Feed do Facebook são veiculados no Instagram, mesmo que usem certas funcionalidades. *No entanto, alguns recursos não são exibidos:*


- [Sobreposições e ferramentas de criação](https://www.facebook.com/business/help/517601898571657)
- [Cartões fixos](https://www.facebook.com/business/help/127241278196607) no final do anúncio


Os anúncios de catálogo Advantage+ para o Feed do Facebook não são veiculados no Instagram Stories se tiverem o seguinte:


- Cartões de mapa
- Vídeos do catálogo
- Cartões de categoria
- Coleções
- O `objective` da campanha `STORE_VISITS`


Cada post do Instagram é gerado de maneira dinâmica. Por isso, [não é possível receber o `instagram_permalink_url` e o `effective_instagram_media_id` do modelo de criativo do anúncio](https://developers.facebook.com/docs/instagram/ads-api/guides/post-moderation#comments-restrictions). Isso significa que você não recebe as curtidas nem os comentários dos posts de anúncios do catálogo Advantage+ no Instagram.
[○](https://developers.facebook.com/docs/instagram/ads-api/guides/advantage-catalog-ads#)
