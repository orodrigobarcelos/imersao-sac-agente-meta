# Noções básicas sobre otimização de anúncios


A API de Marketing oferece pontos de extremidade para gerenciar públicos e analisar insights sobre campanhas publicitárias. Entender esses pontos de extremidade e as respectivas funcionalidades é importante para desenvolvedores iniciantes e experientes que buscam otimizar as estratégias de publicidade.


## Pontos de extremidade para otimização de anúncios


### O ponto de extremidade `customaudiences`


O [ponto de extremidade `customaudiences`](https://developers.facebook.com/docs/marketing-api/reference/ad-account/customaudiences) permite que você crie e gerencie públicos personalizados e semelhantes, adaptando anúncios a segmentos específicos de usuários com base em dados demográficos, interesses e comportamentos.


**Exemplo de solicitação da API**

```
curl -X POST \
  https://graph.facebook.com/v25.0/act_<AD_ACCOUNT_ID>/customaudiences \
  -F 'name=My Custom Audience' \
  -F 'subtype=CUSTOM' \
  -F 'access_token=<ACCESS_TOKEN>'
```


### O ponto de extremidade `insights`


O [ponto de extremidade `insights`](https://developers.facebook.com/docs/marketing-api/reference/ad-account/insights) fornece análises valiosas sobre o desempenho de campanhas, conjuntos de anúncios e anúncios, permitindo que você monitore métricas essenciais como impressões, cliques e conversões.


**Exemplo de solicitação da API**

```
curl -X GET \
  https://graph.facebook.com/v25.0/act_<AD_ACCOUNT_ID>/insights \
  -F 'fields=impressions,clicks,spend' \
  -F 'time_range={"since":"2023-01-01","until":"2023-12-31"}' \
  -F 'access_token=<ACCESS_TOKEN>'
```
