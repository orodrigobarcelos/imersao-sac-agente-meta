# Adicionar elementos interativos


É possível incluir elementos interativos para aumentar o engajamento nos seus anúncios do Instagram.


## Marcações de produto


Você pode adicionar etiquetas de produtos aos seus anúncios, usando o `product_tag_spec` em `interactive_components_spec`. Em `component`, você precisará especificar:


- `type` como `product_tag`
- `product_id` em `product_tag_spec` **Observação:** os produtos precisam ser aprovados antes de receber a marcação.
- Os parâmetros `x` e `y` de `position_spec` **Observação:** os parâmetros `x` e `y` são **obrigatórios** para anúncios com imagem, mas **opcionais** para anúncios com vídeo.


Exemplo de mídia de vídeo/imagem única de feed do `interactive_components_spec`

```
curl \ -F 'name=Sample Creative TESTING' \ -F 'object_story_spec={ "link_data": { "link": "https://www.facebook.com/", "attachment_style": "link", "call_to_action": { "value": {}, "type": "LEARN_MORE" }, "image_hash": "IMAGE_HASH", "image_crops": {}, "preferred_image_tags": {}, "media_elements": {}, "customization_rules_spec": {}, "retailer_item_ids": {}, "collection_thumbnails": {} }, "page_id": "PAGE_ID", "instagram_user_id": "<IG_USER_ID>" } ' \ -F 'interactive_components_spec={ "components": [ { "type": "product_tag", "product_tag_spec": { "product_id": 4382881195057752 }, "position_spec": { "x": 0.5, "y": 0.5, } } ] }' \ -F 'access_token=<ACCESS TOKEN>' \ https://graph.facebook.com/v25.0/act_<AD_ACCOUNT_ID>/adcreatives
```


Exemplo de vários objetos `interactive_components_spec` para mídia de carrossel do feed

```
curl \ -F 'name=Sample Creative TESTING' \ -F 'object_story_spec={ "link_data": { "link": "https://www.facebook.com/", "attachment_style": "link", "call_to_action": { "value": {}, "type": "LEARN_MORE" }, "image_hash": "IMAGE_HASH", "image_crops": {}, "preferred_image_tags": {}, "media_elements": {}, "customization_rules_spec": {}, "retailer_item_ids": {}, "collection_thumbnails": {} }, "page_id": "PAGE_ID", "instagram_user_id": "<IG_USER_ID>" }' \ -F 'interactive_components_spec={ "child_attachments": [ { "components": [ { "type": "product_tag", "position_spec": { "x": 0.5, "y": 0.5 }, "product_tag_spec": { "product_id": 4382881195057752 } } ] }, { "components": [ { "type": "product_tag", "position_spec": { "x": 0.2, "y": 0.8 }, "product_tag_spec": { "product_id": 4382881195057752 } } ] }, { "components": [ { "type": "product_tag", "position_spec": { "x": 0.3, "y": 0.7 }, "product_tag_spec": { "product_id": 4382881195057752 } } ] } ] }'\ -F 'access_token=<ACCESS TOKEN>' \ https://graph.facebook.com/v25.0/act<AD_ACCOUNT_ID>/adcreatives
```


### Ler as informações da etiqueta de produto


Você pode acessar informações sobre o `product_id` no seu objeto `product_tag_spec` a partir de um [criativo do anúncio](https://developers.facebook.com/docs/marketing-api/reference/ad-creative) criado com o tipo `product_tag`.


Para recuperar as informações de `product_tag_spec` do criativo do anúncio:

```
curl -G \ -d "fields=interactive_components_spec" \ -d "access_token=<ACCESS_TOKEN>" \ "https://graph.facebook.com/v25.0/<AD_CREATIVE_ID>"
```


Exemplo de resposta

```
{ "interactive_components_spec": { "components": [ { "position_spec": { "x": 0.5, "y": 0.5 }, "type": "PRODUCT_TAG", "product_tag_spec": { "product_id": "4382881195057752" } } ] }, "id": "23862467608880374" }
```


### Marcação automatizada para anúncios estáticos


Este recurso marca automaticamente os produtos do seu catálogo com melhor correspondência durante a criação de anúncios com imagens únicas ou imagens de carrossel carregadas; depois ele aplica as etiquetas de produto relevantes nos anúncios.


O sistema de marcação automatizada de anúncios detecta correspondências de duas formas:


- Imagens no seu catálogo: se os recursos do criativo do seu anúncio corresponderem aos das imagens de produtos do seu catálogo, o sistema de anúncios adicionará automaticamente a etiqueta de produto ao criativo.
- URL do site no destino do anúncio: se o URL do site inserido na seção de destino no nível do anúncio corresponder aos itens do seu catálogo, o sistema de anúncios adicionará automaticamente a etiqueta de produto ao criativo do seu anúncio.


Se você especificar etiquetas de produto no parâmetro `interactive_components_spec`, o recurso de marcação automatizada será desabilitado para que o anúncio respeite as etiquetas especificadas.


**Observação:** esse recurso é habilitado por padrão. Defina `enroll_status` como `opt_out` no parâmetro `interactive_components_spec` para desativar o recurso por item de produto.

```
"interactive_components_spec": { "components": [ { "type": "product_tag", "enroll_status": "opt_out" } ] }
```


**Observação:** você não pode definir `enroll_status` e `product_tag_spec` ao mesmo tempo, pois retornará um erro.
