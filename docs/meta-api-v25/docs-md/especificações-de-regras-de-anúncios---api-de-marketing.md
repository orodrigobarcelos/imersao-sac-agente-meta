# Especificações de regras de anúncios


As especificações de regras de anúncios determinam as ações que devem ser executadas e os objetos sobre os quais essas ações devem ser aplicadas.


### [Especificação de avaliação](https://developers.facebook.com/docs/marketing-api/ad-rules/overview/evaluation-spec)


Determina os objetos sobre os quais a regra deve executar a ação.


### [Especificação de execução](https://developers.facebook.com/docs/marketing-api/ad-rules/overview/execution-spec)


Determina a ação que se aplica a todos os objetos que passam na avaliação.


### [Especificação de alteração](https://developers.facebook.com/docs/marketing-api/ad-rules/overview/change-spec)


É a opção usada para tipos de execução, como `CHANGE_BUDGET` e `CHANGE_BID`.
