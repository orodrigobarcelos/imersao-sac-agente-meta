# Adicionar chamada para ação


Como opção, você pode fornecer uma chamada para ação (CTA) no `link_data` ao [criar seus anúncios no Instagram](https://developers.facebook.com/docs/marketing-api/using-the-api/get-started). Se você não informar uma CTA, o padrão será `LEARN_MORE` como o `type`, e o `link` de `link_data` como o `link`. Nem todos os tipos podem ser usados para todos os anúncios. Estamos introduzindo aos poucos um novo conjunto de seis objetivos de campanha no Gerenciador de Anúncios: reconhecimento, tráfego, engajamento, leads, promoção do app e vendas. Consulte o [Guia de produtos de anúncios](https://www.facebook.com/business/ads-guide/update) para ver qual tipo pode ser usado com base no `objective` da sua campanha.


## Opções


Você não deve criar anúncios `WEBSITE_CLICKS` que sejam vinculados à sua Página do Facebook nem ao perfil do Instagram. Além de poder resultar em um erro na API, também cria uma experiência insatisfatória de publicidade. Por exemplo, se você vincular esse tipo de anúncio ao seu perfil do Instagram, todos os usuários que não estiverem conectados por meio do navegador no app verão um diálogo de início de sessão novamente. Para promover sua Página do Facebook com um anúncio do Facebook, dê preferência para anúncios `PAGE_LIKES`.
