# Criativo Advantage+


O criativo Advantage+ é destinado a anúncios que usam imagem única, vídeo único, carrossel, catálogo ou publicação existente. Eles criam automaticamente diversas variações do anúncio e mostram uma versão personalizada para cada conta da Central de Contas com base na probabilidade de repercussão.


Você pode criar anúncios com o criativo Advantage+ para ajudar a impulsionar o desempenho e veicular anúncios mais personalizados para cada conta da Central de Contas.


Para saber mais, consulte [Sobre o criativo Advantage+](https://www.facebook.com/business/help/297506218282224).
