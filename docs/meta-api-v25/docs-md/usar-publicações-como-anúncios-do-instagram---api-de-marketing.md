# Usar posts como anúncios do Instagram


Use posts do Instagram ou do Facebook para criar anúncios no Instagram.


## Limitações


- Não é possível turbinar posts de mídia contendo músicas protegidas por direitos autorais ou elementos interativos como filtros.
[○](https://developers.facebook.com/docs/instagram/ads-api/guides/use-posts-as-ads#)

## Posts do Instagram


Crie anúncios a partir de posts do Feed do Instagram com vídeos, carrossel, vídeos do Reels ou fotos individuais (fotos e vídeos com marcação são compatíveis) ou stories ativos do Instagram. Você também pode criar anúncios a partir de posts do feed que contenham imagens do feed, carrosséis e vídeos existentes com etiquetas de produto. Atualmente, não é possível usar posts do Instagram TV para criar anúncios.


### Etapa 1: obtenha o número de identificação do usuário do Instagram


Você pode obter o número de identificação do usuário do Instagram (usuário do Instagram) por meio da [API do Instagram](https://developers.facebook.com/docs/instagram-api/reference/ig-user/)


#### Graph API do Instagram


Para obter o número de identificação do usuário de uma conta empresarial ou de criador de conteúdo do Instagram, use o campo `instagram_business_account` na Página relevante do Facebook. Consulte [Graph API do Instagram: Introdução](https://developers.facebook.com/docs/instagram-api/getting-started).


#### API de Anúncios do Instagram


Use a API de Marketing para obter o número de identificação do usuário de uma conta do Instagram conectada a uma conta de anúncios ou a uma Página específica. Para receber essas informações, consulte os pontos de extremidade a seguir:


- [`{ad_account_id}/connected_instagram_accounts`](https://developers.facebook.com/docs/marketing-api/reference/ad-account/connected_instagram_accounts/): obtenha o número de identificação do usuário de contas do Instagram conectadas a uma conta de anúncios.
- [`{business_id}/instagram_business_accounts`](https://developers.facebook.com/docs/marketing-api/reference/business/instagram_business_accounts/): receba o número de identificação do usuário de contas conectadas a uma empresa.


Caso você precise conectar sua conta do Instagram, consulte [Como adicionar uma conta do Instagram ao Gerenciador de Negócios](https://www.facebook.com/business/help/620548115562686) e [O que acontece ao conectar sua conta do Instagram e uma Página do Facebook que você gerencia](https://l.facebook.com/l.php?u=https%3A%2F%2Fhelp.instagram.com%2F402748553849926&h=AT4DLCMvUxkiHpZRHv0ggl3oiH_E3UxHFSYbtGULr_5OA4aOV8dDRhYUu9ah6wkvRAon02_m-31bp7xDNvcqBbhjluDnFj4nNxhaieC_iK9aAoqpdjETQkLfeYNrFXl8lgBbIpQaAare9eeksq2aK_KGJRSVV5R2xQJYl-lz).


### Etapa 2: encontre o post que você quer promover e verifique a qualificação


Para obter a identificação do post relevante do Instagram ([Mídia do Instagram](https://developers.facebook.com/docs/instagram-api/reference/ig-media)) que deseja usar como anúncio, utilize os [pontos de extremidade de mídia](https://developers.facebook.com/docs/instagram-api/reference/ig-user/media) da Graph API do Instagram. Para obter o número de identificação do story relevante do Instagram que deseja usar como anúncio, use o [ponto de extremidade dos stories](https://developers.facebook.com/docs/instagram-api/reference/ig-user/stories) da Graph API do Instagram. Esse número será usado como o `source_instagram_media_id` do anúncio.


Você pode usar o campo `boost_eligibility_info` para confirmar se a mídia tem qualificação para ser turbinada como um anúncio.


Se você já tiver configurado um criativo do anúncio com um post do Instagram, verifique o número de identificação original da mídia consultando `{ad_creative_id}/source_instagram_media_id`.


### Etapa 3: crie anúncios


Para criar anúncios, conclua o processo descrito em [API de Anúncios do Instagram: Introdução](https://developers.facebook.com/docs/marketing-api/guides/instagramads/get-started) até a **Etapa 3: crie um conjunto de anúncios**. Na **Etapa 4: forneça um criativo**, faça estas alterações:


- Em vez de especificar `instagram_user_id` no criativo, defina `instagram_user_id` com o nome de identificação do usuário do Instagram recebido na [Etapa 1: obtenha o número de identificação do usuário do Instagram](https://developers.facebook.com/docs/instagram/ads-api/guides/use-posts-as-ads#ig-posts-step-1). Por exemplo: ``` { "object_id":"<object_id>", // page id "instagram_user_id":"<IG_USER_ID>", "source_instagram_media_id":"<source_instagram_media_id>" } ```
- Especifique `source_instagram_media_id` como o ID de mídia obtido na [Etapa 2: encontre a publicação que você quer promover](https://developers.facebook.com/docs/instagram/ads-api/guides/use-posts-as-ads#ig-posts-step-2). Por exemplo: ``` curl -i -X POST \ "https://graph.facebook.com/v<API_VERSION>/act_<AD_ACCOUNT>/adcreatives ?object_id=<PAGE_ID> &instagram_user_id=<IG_USER_ID> &source_instagram_media_id=<IG_ORGANIC_MEDIA_ID> &access_token=<API_ACCESS_TOKEN>" ```
- Se desejar, atualize o [campo `call_to_action`](https://developers.facebook.com/docs/instagram/ads-api/reference/data-cta-requirements) para sua promoção. Por exemplo: ``` curl -i -X POST \ "https://graph.facebook.com/v<API_VERSION>/act_<AD_ACCOUNT>/adcreatives ?object_id=<PAGE_ID> &instagram_user_id=<IG_USER_ID> &source_instagram_media_id=<IG_ORGANIC_MEDIA_ID> &call_to_action="{'type':'LEARN_MORE','value':{'link': '<YOUR_LINK>'}}" &access_token=<API_ACCESS_TOKEN>" ``` #### Chamada para ação Você pode definir a chamada para ação dependendo do destino: [Instagram](https://developers.facebook.com/docs/marketing-api/ad-creative/messaging-ads/click-to-instagram), [Messenger](https://developers.facebook.com/docs/marketing-api/ad-creative/messaging-ads/click-to-messenger) ou [vários destinos](https://developers.facebook.com/docs/marketing-api/ad-creative/messaging-ads/click-to-multidestination). ##### Clique para o Instagram ``` "call_to_action": { "value": {"app_destination":"INSTAGRAM_DIRECT"}, "type": "MESSAGE_PAGE" } ``` ##### Clique para o Messenger ``` "call_to_action": { "value": {"app_destination":"MESSENGER"}, "type": "MESSAGE_PAGE" } ``` ##### Clique para vários destinos ``` "asset_feed_spec": { "optimization_type": "DOF_MESSAGING_DESTINATION", "call_to_actions": [ { "type": "MESSAGE_PAGE", "value": { "app_destination": "MESSENGER", "link": "https://fb.com/messenger_doc/" } }, { "type": "INSTAGRAM_MESSAGE", "value": { "app_destination": "INSTAGRAM_DIRECT", "link": "https://www.instagram.com" } } ] } ``` Consulte [Especificação do feed de ativos](https://developers.facebook.com/docs/marketing-api/ad-creative/asset-feed-spec) para ver mais informações.


Depois de fornecer o criativo, [crie seus anúncios](https://developers.facebook.com/docs/marketing-apis/get-started#book-ad). Também é possível fornecer o criativo durante a geração dos anúncios, sem separar o processo em duas etapas.
[○](https://developers.facebook.com/docs/instagram/ads-api/guides/use-posts-as-ads#)

## Posts do Facebook


Nem todas as publicações do Facebook funcionam como anúncios do Instagram.


Você pode usar uma publicação no Feed do Facebook ou um story como parte do criativo do anúncio para impulsionar o seu conteúdo do Instagram. Faça esta chamada de API para saber se um post pode ser usado em anúncios:

```
curl -G \ -d "access_token=<ACCESS_TOKEN>"\ -d "fields=is_instagram_eligible"\ "https://graph.facebook.com/<API_VERSION>/<POST_ID>"
```


Se a resposta incluir `"is_instagram_eligible": true`, você poderá iniciar a criação do anúncio.


### Chamada para ação


```
curl -i -X POST \ "https://graph.facebook.com/v25.0/act_<AD_ACCOUNT>/adcreatives ?object_story_id=<postOwnerID_postID> &instagram_user_id=<IG_USER_ID> &call_to_action="{'type':MESSAGE_PAGE,'value':{'app_destination':'MESSENGER'}}" &access_token=<ACCESS_TOKEN>"
```


`object_story_id` é a identificação do post no formato `postOwnerID_postID`, e `instagram_user_id` é uma identificação da conta do Instagram conectada à Página ou a identificação da conta do Instagram associada à Página. Veja mais detalhes em [Configurar contas do Instagram com Páginas](https://developers.facebook.com/docs/instagram/ads-api/guides/pages-ig-account).
[○](https://developers.facebook.com/docs/instagram/ads-api/guides/use-posts-as-ads#)

## Solução de problemas


- Caso você veja o erro "O criativo deve fornecer enroll_status para aprimoramentos padrão", consulte [Aprimoramentos padrão no criativo Advantage+](https://developers.facebook.com/docs/marketing-api/advantage-catalog-ads/standard-enhancements) para saber mais.
- Se estiver promovendo um post do Facebook com um `instagram_user_id` no criativo do anúncio e o conjunto de anúncios incluir posicionamentos do Facebook e do Instagram, talvez você veja uma das seguintes mensagens de erro: "O criativo não tem especificações de DOF" ou "O criativo deve ter especificações degrees_of_freedom para anúncios com vários destinos". Adicione `"optimization_type": "DOF_MESSAGING_DESTINATION"` ao parâmetro `asset_feed_spec` no criativo do anúncio.
[○](https://developers.facebook.com/docs/instagram/ads-api/guides/use-posts-as-ads#)
